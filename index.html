Practice Makes Perfect

  1. Create a new repository or repo in Github.com
  2. Name the repo
  3. Add a Readme
  4. Clone or download it by copying it
  5. Open up Gitbash or terminal and choose where to create or download it by using cd and then the directory/folder to clone it in
  6. Then git clone paste the copied link
  7. Create html root file (index.html) in that folder
  8. Then add some code to that file by opening it in visual studio code
  9. Then git add -A
  10. Now your html file is being tracked
  11. Then git commit -am "myfile"
  12. then git push
  13. then go the link https://whatever you named the repo
